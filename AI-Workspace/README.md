# AI Workspace

This vault is a small working memory for AI-assisted development.

Use it for:

- Project rules that Codex and Claude should reuse.
- Prompts that worked well.
- Debug logs and root causes.
- Design/system notes before turning them into Claude Code skills.
- Decisions that should not be rediscovered every session.

## Folders

- `00_Inbox`: Quick captures before sorting.
- `01_Projects`: One note per repo or product.
- `02_Prompts`: Reusable prompts for Codex, Claude, and other agents.
- `03_Skills`: Drafts for Claude/Codex skills.
- `04_References`: API notes, docs summaries, and implementation references.
- `05_Decisions`: Architecture and workflow decisions.
- `06_Debug-Logs`: Errors, fixes, and prevention notes.
- `Templates`: Obsidian note templates.

## Daily Use

1. Keep the active project note current.
2. Save good prompts after they work twice.
3. Write a debug log when a bug took more than 10 minutes.
4. Ask the agent to read the project note before large edits.
