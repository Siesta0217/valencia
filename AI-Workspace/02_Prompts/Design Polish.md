# Design Polish

## Use When

Use this for UI surfaces such as ClickGUI, HUDs, overlays, settings panels, and visual QA.

## Prompt

```text
Please audit this UI with a design-review mindset.

Focus on:
- visual hierarchy
- spacing and alignment
- readable typography
- color contrast
- interaction states
- mobile/window scaling if relevant
- avoiding clutter and generic decoration

Keep existing product identity intact. Suggest concrete changes first, then implement the smallest useful polish pass.

Target:
```

## Notes

For this Java mod, design work likely means HUD readability, ClickGUI controls, color choices, and avoiding overlap.
